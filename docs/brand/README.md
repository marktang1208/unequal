# 不等号 / unequal — 头像设计

小程序头像源文件。1024×1024 正方形源稿（PNG with alpha），微信小程序头像会自动按圆形裁剪显示，所以源稿四角透明、四面留白足够。

## 候选方向（三个版本）

> 均为 1024×1024 方形源文件；预览缩略图时微信会按圆形裁剪。

| 版本 | 文件 | 调性 | 配色 |
|---|---|---|---|
| A. 温暖拥抱 | `avatar-warm.png` | 温度感、亲和、专业 | 桃粉 → 珊瑚橙，奶白符号 |
| B. 理性绿洲 | `avatar-cool.png` | 理性、可信、知识库 | 薄荷绿 → 蓝绿，深绿符号 |
| C. 童趣奶昔 | `avatar-cute.png` | 童趣、有温度、年轻 | 柔粉 → 桃粉，玫红符号 + 暖黄小心 |

## 设计要点

- 主体：粗笔画圆角的「≠」符号（白/深色，两条反向倾斜的等粗笔画），在小尺寸（64px 列表头像）下依然清晰可辨。
- 留白：主符号四周至少留出约 18% 的安全边距，避免被系统圆角切掉或在小尺寸时显得拥挤。
- 字号：笔画粗细 ≈ 边长 10%，长度 ≈ 边长 56%，间距 ≈ 边长 12%。
- 倾斜：±20° 左右对称，给人"动态但稳定"的视觉。

## 复现 / 二次修改

```bash
python3 docs/brand/_drafts.py
```

调色和几何参数都在 `docs/brand/_drafts.py` 顶部的 `design_warm/cool/cute()` 里。
