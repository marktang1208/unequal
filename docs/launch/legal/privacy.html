隐私政策

最后更新: 2026-06-26 · 适用产品: 微信小程序「育儿不等号」

一、我们收集的信息

1.1 您主动提供的信息

  • 您上传/抓取的育儿相关文档（PDF、Word、TXT、Markdown、网页内容）
  • 您的问答历史（提问内容、AI 回答、引用来源）

1.2 微信小程序自动采集的信息

  • 微信 OpenID（用于会话识别, 由微信云开发生成, 我们无法直接获取您的微信号）
  • 设备信息: 设备型号、操作系统版本（仅用于排错）
  • 网络信息: IP 地址（仅用于请求路由和反滥用）

1.3 我们不收集的信息

  • ❌ 您的真实姓名、身份证号、手机号
  • ❌ 您的位置信息（GPS / IP 定位）
  • ❌ 您的通讯录、相册、麦克风
  • ❌ 您的微信好友关系

二、信息使用目的

数据用途存储位置
OpenID标识用户会话微信云开发 D1 数据库
问答历史多轮会话 + 历史记录查询微信云开发 D1 数据库
上传文档知识库构建 + 检索增强生成微信云开发 R2 + 向量数据库
设备/网络信息排错、反滥用微信云开发日志

三、信息共享

我们不会主动向第三方分享您的个人信息, 除非:

  • 获得您的明确同意
  • 法律法规或行政机关要求
  • 仅共享给必要的服务供应商:
    • 微信云开发 (CloudBase): 提供存储、计算基础设施
    • MiniMax API: 提供 LLM 推理和 embedding 服务 (仅传输问答内容, 不含用户身份)

四、信息存储

  • 存储位置: 腾讯云上海地域 (ap-shanghai)
  • 存储期限: 您删除小程序账号后, 我们将在 30 天内删除您的数据
  • 数据出境: 不出境

五、您的权利

您有权:

  • 查询: 通过小程序「设置」页查看 user_id 和会话列表
  • 删除: 通过小程序「设置」页请求删除账户（开发中）
  • 更正: 通过小程序「设置」页反馈错误内容

六、未成年人保护

本服务面向家长群体, 14 周岁以下儿童不应直接使用本服务。如发现儿童直接使用, 家长应及时删除相关问答历史。

七、本政策更新

  • 重大变更将通过小程序内通知
  • 继续使用本服务即视为接受更新后的政策
