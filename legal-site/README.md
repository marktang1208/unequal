# 不等号 / unequal — 用户协议 / 隐私政策

本仓库托管微信小程序「育儿不等号」的法定文档。

## 在线访问

- 用户协议: <https://marktang1208.github.io/unequal/>
- 隐私政策: <https://marktang1208.github.io/unequal/privacy.html>

> 注: 当前部署在 marktang1208/unequal repo 的 gh-pages 分支(不是独立的 unequal-legal repo)。

## 文件说明

- `index.html` — 用户协议
- `privacy.html` — 隐私政策

## 部署

本仓库使用 GitHub Pages 部署（公开仓库，免费，自动 HTTPS）。

部署步骤参见父项目: `../docs/launch/02-github-pages-setup.md`

## 更新文档

修改完 `index.html` / `privacy.html` 后:

```bash
git add .
git commit -m "docs: update legal terms"
git push origin main
```

GitHub Pages 1-2 分钟内自动更新。
